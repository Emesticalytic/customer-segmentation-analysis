# Customer Segmentation Analysis

This project is a customer segmentation analysis project for a data analytics and data science portfolio.

The aim is to understand customer behaviour, revenue value and retention risk using retail transaction data.

## Business Question

```text
How can we group customers into meaningful segments so the business can improve marketing, retention and revenue?
```

## Project Scope

This project will be built step by step:

1. Project setup
2. Dataset preparation
3. Data cleaning and manipulation
4. DuckDB SQL joins
5. Exploratory data analysis
6. RFM analysis
7. Customer segmentation
8. Revenue analysis
9. Statistical analysis and A/B testing
10. Record leakage checks
11. Baseline machine learning model
12. Backend API
13. Frontend dashboard
14. Final documentation
